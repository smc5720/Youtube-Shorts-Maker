# Youtube-Shorts-Maker
유튜브 쇼츠 제작을 위한 자동 생성 엔진과 세로 영상 편집 앱

## 기획

- [PRD](docs/PRD.md): 주제, 원문, 기사 전문, 링크 입력 기반 YouTube Shorts 자동 생성 MVP 기획서

## MVP 방향

1차 목표는 YouTube 자동 업로드가 아니라 `주제/원문/링크 입력 -> 요약 -> 대본 -> 음성 -> 자막 -> 편집 -> 1080x1920 MP4` 제작 앱입니다. 사용자는 생성된 초안을 앱에서 열고 자막, 텍스트, 배경, 장면 순서를 수정한 뒤 최종 영상을 내보냅니다. 업로드 자동화는 YouTube Data API 정책과 OAuth 구성을 별도 검토한 뒤 2차 기능으로 진행합니다.
